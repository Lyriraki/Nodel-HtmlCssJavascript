Apa Itu NodeJS?

Node.js adalah platform buatan Ryan Dahl untuk menjalankan aplikasi web berbasis JavaScript yang dikenalkan pada tahun 2009. Dengan platform ini, Anda dapat menjalankan JavaScript dari sisi server. Untuk mendukung kemampuan tersebut, Node.js dibangun dengan engine Javascript V8 milik Google.
